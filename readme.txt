There are two code files provided in this folder.

All the required packages for installation are given in the requirements.txt file

bertImplementation.py is used to run a BERT model on a dataset. It contains the required code for taking input a dataset and converting it into a proper format required by BERT.
Further, it runs the model and gives output the various metrics.

dataAugmentation.py is used to run the augmentation algorithm on a given dataset. It contains the code of the different modules of the algorithm and finally saves a new csv file with the updated data points

Some examples of the working of the augmentation are shown in the examples.pdf
